# Unmix

## Examples:

```
./unmix [ source type ] -i [ input ] [ output ]
```

Command line tool to: 
1. Download, 
2. cut
3. rename files
4. move to a 'unmixed' compilation folder on your desktop

## Dependencies
1. youtube-dl (always latest!)
2. ffmpeg (v2+)
